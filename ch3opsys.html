Operating Systems Terminology
Terms Definitions
Operating System is a software program that manages a computers hardware and software resources.
Job Queue a data structure that is mantained by job scheduler software contatins jobs.
FIFO is First-in,First-out; which is a method of organizing and manipulating data in a specific order.
Interactive Processing is a process that allows users to interact directly with a computer program or data on a screen.
Time-sharing a computer technique that allows multiple users or processes to share a computer's resources simultaneously.
Multiprogramming is a type of multitasking that allows multiple programs to run at the same time ona single CPU.
Application Software a computer program that helps users complete specific tasks, such as productivity, communication, or creativity.
System Software a collection of programs that manage and control a computer's hardware and applications.
Utility Software a program that helps maintain and improve a computer's performance, fuctionality, and efficiency.
Graphical User Interface (GUI) a visual way for users to interact with electronic devices, such as computers and software.
Shell a program that acts as an intermediary between a user and an operating system's kernel.
Kernel is the core of an operating system.
Device Driver is a software program that allows a computer's operating system and applications to communicate with hardware devices.
File Manager displays the file/folders hierarchy; the main page to a folder/file.
Memory Manager a system that controls how a computer's memory is used.
Virtual Memory a memory management technique that simulates RAM and gives the impression of a large main memory.
Boot Strapping to boot or to load a program into a computer using a much smalller intial program to load the desired program.
ROM means Read-Only memory, which is a type of computer memory that stores permanent data.
Deadlock is a situation where two or more processes are unable to make progress because they are waiting for each other to release a shared resource.
Spooling it manages input and output operations by temporarily storing data for later processing or printing.
Super User (or administrator) a user account with the highest level of access in a system or application.
Multi-Factor Authentication a security system that requires users to provide more than one form of authentication.
