Terms Definitions
Machine Language the lowest level of programming language that directly corresponds to the instructions executed by a computer's hardware.
Debugging: is the process of fixing errors or "bugs" within a software program.
Assembler: a computer program that translates assembly language into machine code.
Assembly Language: a type of low-level programmming language that is intended to communitcate directly with a computers hardware. Example, Binary and hexadecimal characters.
Translator: a program that converts source code into a form that a computer processor can execute.
Compiler: a software tool that translates source code written in a high-level programming lanaguage into a low-level language.
Interpreter: a program that executes instructions in a programming language without first compiling them into machine language.
Cross-platform Software: a computer product or system compatible with several operating systems.
Natural Language: a field of computer science that uses machine learning to teach computers to understand, manipulate, and interpret human language.
Formal Language: a set of strings of symbols that are defined by rules and recongnized by a formal machine.
Machine Independence: a program can run on many different types of computers, or computer architectures.
High-level Programming Language: a programming language that is designed to be easier for humans to understand and write.
Object-oriented Programming (OOP) a computer programming model that organizes software around objects, rather than logic and functions.
Scripting Language: a type of programming language that's used to write instructions for a run-time environment
Data Types ************
*Integer: stores whole numbers, or numbers without a decimal point, and can be positive or negative.
*Float: is a numerical data type that allows for representing real numbers with decimal points.
*String: represents a sequence of characters, words, or symbols as a single unit.
Primitive Date Type: a basic data type that is built into a programming language and represents simple values.
Date Structure: a way to organize and store data in a computer so that it can accessed, processed, and retrieved quickly and efficiently.
Phython Collection Date Structures *********************
*List: a collection which is ordered and changeable.
*Tuple: represents and ordered, immutable collection of elements
*Set: abstract data type that represents a collection of unique elements or unordered collection.
*Dictionary: stores data in key-value pairs
Function: a "self contained" modules of code that accomplish a specific task.
Local Variable: a variables that is declared within a specific funtion or code block and can only be used withinb that function or block
Global Variable: a variable that can be accessed from any part of a program
Source Program: a set of instructions written in a programming language that defines how a software application works
Object Program: a machine-readable file that contains instructions for a computer's processor to execute
